# Orys progress log

Use this file to record what was changed in each loop/task run.

## Suggested format

### Task X - <name>

- Issues found and fixed:
  1. `path/to/file.ts:line` - what was wrong and how it was fixed.
- Issues noted (no fix needed):
  1. `path/to/file.ts:line` - why no change was needed.
- Verification:
  - Commands run and outcome.
